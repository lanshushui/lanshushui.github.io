unordered_map 用 [] 访问，但 key 不存在时，访问仍然成功，取得 value 对象默认构造的值

enum 可以直接作为 map 的 key，却不能直接作为 unordered_map 的 key。

value 不能是抽象类，针对这场景需要用到指针或者智能指针 shared_ptr
